# **LBS Practice Repository**
# **Project Title**
LBS Practice – Git Branching Assignment

# **Project Description**
This repository is created as part of the Git & GitHub Assignment to demonstrate the use of 
Git version control concepts such as branch creation, commits, and pushing code to GitHub using Eclipse IDE.
The repository contains Java programs organized into different branches to represent different days of practice work.
- branch1 contains Day1 programs
- branch2 contains Day2 programs
This assignment helps in understanding how developers manage different versions of code using branches in Git.

# **Technologies Used**
-Java
-Git
-GitHub
-Eclipse IDE

## **Git Concepts Demonstrated**
This repository demonstrates the following Git operations:
- Repository Creation
- Cloning Repository
- Adding Files
- Committing Changes
- Creating Branches
- Switching Branches
- Pushing Code to GitHub
